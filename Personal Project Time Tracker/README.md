# ⏳ Personal Project Time Tracker - C# WinForms (v3.0)



## 📝 عن المشروع (About)
مشروع **Time Tracker** هو تطبيق لسطح المكتب تم بناؤه باستخدام **C# WinForms**. يساعدك التطبيق على إدارة وقتك من خلال تتبع الدقائق والثواني المستغرقة في تنفيذ المهام داخل مشاريعك المختلفة.

هذا المشروع هو ثمرة تطبيقية لما تعلمته في مسار الدكتور [**محمد أبو هدهود**](https://programmingadvices.com/) عبر منصة **Programming Advices**، وتحديداً بعد إتمام الكورسات المتقدمة في قواعد البيانات وهيكلة المشاريع.

هذا المشروع جزء من رحلتي في قناة **"Build With Ibrahim"** لتوثيق بناء المشاريع البرمجية.

## 🚀 سجل التحديثات (Versions History)

### **V3.0 (الأحدث - النسخة الحالية)** 💎
* **قاعدة بيانات SQL Server:** الانتقال من نظام الملفات (File System) إلى قاعدة بيانات حقيقية لضمان استقرار وسرعة البيانات.
* **هيكلة الطبقات الثلاث (3-Tier Architecture):** تم فصل المشروع بالكامل إلى:
  - **DataAccessLayer:** للتعامل مع قاعدة البيانات.
  - **BusinessServices:** لمنطق العمل (Logic).
  - **Presentation Layer (UI):** واجهة المستخدم (WinForms).
* **إصلاحات تقنية:** تحسين كود التعامل مع البيانات وضمان عدم ضياع التوقيت عند الإغلاق المفاجئ.

### **V2.0**
* **حذف المشاريع:** إضافة خاصية الحذف باستخدام ID المشروع.
* **تعديل المشاريع:** إمكانية تعديل الوصف والوقت المتوقع للمشاريع القائمة.

### **V1.0 (البداية)**
* **إدارة المهام:** إضافة مشاريع وربط مهام فرعية بها.
* **عداد حي (Live Timer):** تتبع الوقت مع إمكانية البدء والإيقاف.
* **نظام التنبيهات:** إشعار عند الوصول للوقت المتوقع.

## 🛠 التقنيات المستخدمة (Tech Stack)
* **اللغة:** C#
* **قاعدة البيانات:** SQL Server
* **النمط المعماري:** 3-Tier Architecture (Clean Code Principles)
* **البيئة:** .NET Framework / Windows Forms

## ⚙️ كيفية التشغيل (How to Run)
1. قم بعمل `Clone` للمستودع.
2. تأكد من وجود **SQL Server** مثبت على جهازك.
3.قم بانشاء قاعدة بيانات وعدل  `Connection String` في طبقة الـ Data Access
4. افتح ملف الـ `.sln` باستخدام **Visual Studio 2022**.
5. اضغط **F5** للتشغيل.
